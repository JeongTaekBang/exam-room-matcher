10. 날짜 → 시트 매핑 = 시험일을 격자로 연결

시험일자(날짜)가 어떤 시간표 시트(=격자)에 대응되는지를 연결하는 과정.

엑셀 시트 이름 → 파싱 → 날짜 → 요일
"4.21.(화)" → 월=4, 일=21, 요일=화 → 2026-04-21 → 화
"4.22.(수)" → 월=4, 일=22, 요일=수 → 2026-04-22 → 수
"4.23.(목)" → 월=4, 일=23, 요일=목 → 2026-04-23 → 목

한 장 요약

모든 산수 = 격자 위에서 셀을 채우고, 비우고, 세기

  • 분류구조화된 컬럼 기반 5가지 분기 판정
  • 매핑시험일자 → 요일 → 시트 → 격자 연결
  • 검수격자에 올리기 전 데이터 품질 확인
  • 배정빈 셀에 시험 넣기
  • 해제미실시 / 이동 / 부분 → 셀 비우기
  • 충돌한 셀에 2개 이상 → 빨간 경고
  • 빈 강의실비었거나 해제된 셀 찾기
  • 수급열(교시)별 빈 셀 수 vs 수요
  • 분반 완료수용인원 합 >= 수강생 수
  • 가동률찬 셀 / 전체 셀 x 100
