# 강의실 배정 프로그램 — 산수 가이드

이 문서는 프로그램에서 사용되는 모든 계산과 판정 로직을 직관적으로 설명한다.

---

## 1. 시간의 기본 단위: 교시

하루는 **15개 교시**(0~14교시)로 나뉜다.

```
교시   시간
 0    8:00 ~ 9:00
 1    9:00 ~ 10:00
 2    10:00 ~ 11:00
 ...
14    22:00 ~ 23:00
```

**변환 공식**: `시작시각 = 8 + 교시`, `종료시각 = 9 + 교시`

모든 교시 값은 0~14 범위로 **클램프**된다 (범위 밖이면 가장 가까운 경계값으로 고정).

---

## 2. 시험 교시 계산

> "이 과목은 몇 교시에 시험을 보는가?"

두 가지 경로로 결정된다:

### 경로 A: 시험 시작/종료 교시가 명시된 경우

```
시험 시작 = 3교시, 시험 종료 = 5교시
→ 시험 교시 = [3, 4, 5]
```

작은 쪽 ~ 큰 쪽까지 연속 범위. 시작/종료가 뒤바뀌어도 자동 정렬한다.

### 경로 B: 수업시간표에서 추출하는 경우

시험일 **요일**에 해당하는 수업 슬롯을 찾아 교시를 합친다.

```
수업시간표: "화 4~6(K106), 수 1~2(K106)"
시험일: 화요일
→ 시험 교시 = [4, 5, 6]
```

여러 슬롯이 있으면 교시를 **합집합**한다.

---

## 3. 강의실 결정

> "이 과목은 어느 강의실에서 시험을 보는가?"

수업시간표에서 **시험 요일**과 같은 요일의 슬롯을 찾아 그 강의실을 사용한다.

```
수업시간표: "화 4~6(K106), 수 1~2(K201)"
시험일: 화요일
→ 시험 강의실 = K106
```

일치하는 슬롯이 없으면 엑셀의 강의실 열 값을 폴백으로 사용.

---

## 4. 분류 판정 (5가지)

각 시험 요청은 엑셀 데이터 기반으로 5가지 중 하나로 분류된다:

| 분류 | 의미 | 판정 기준 |
|------|------|-----------|
| **시험 진행** (NORMAL_EXAM) | 원래 강의실에서 정상 시험 | 강의실 선택 없음 + 시험일자 있음 + 미실시 키워드 없음 |
| **미실시** (NO_EXAM) | 시험 안 봄 (대체과제 등) | 요청사항에 "미실시"/"대체과제" 등 키워드 |
| **강의실 변경** (ROOM_CHANGE) | 다른 강의실로 옮김 | 강의실 선택 있음 + 분반 아님 |
| **강의실 분반** (ROOM_SPLIT) | 여러 강의실로 나눔 | 강의실 선택 있음 + 분반 조건 |
| **미확정** (SKIP) | 정보 부족 | 위 어디에도 해당 안 됨 |

## 5. 자동 해제

> "시험 때문에 비게 되는 강의실·교시를 자동으로 찾아낸다."

세 가지 상황에서 자동 해제가 발생한다:

### 5-1. 미실시 (NO_EXAM) → 전체 해제

시험을 안 보면 해당 수업 시간 전체가 비어난다.

```
과목A: 화 3~5교시 K106, 시험 미실시
→ 해제: K106의 3, 4, 5교시 전부 (라벨: "미실시")
```

### 5-2. 강의실 변경 (ROOM_CHANGE) → 원래 강의실 해제

다른 방으로 옮기면 원래 방이 비어난다.

```
과목B: 원래 K106 → K201로 이동 배정됨
→ 해제: K106의 해당 교시 (라벨: "이동")
```

분반에서 "기존 미유지"를 선택한 경우에도 동일하게 적용.

### 5-3. 시험 진행 (NORMAL_EXAM) → 부분 해제

수업은 3교시인데 시험은 1교시만 보면, 나머지 2교시가 비어난다.

```
과목C: 수업 1~3교시, 시험 1교시만
→ 시험 교시 = [1], 수업 교시 = [1, 2, 3]
→ 해제: 2교시, 3교시 (라벨: "부분해제")
```

**계산**: 수업 교시 - 시험 교시 = 해제 교시

## 6. 충돌 감지

> "같은 강의실, 같은 교시에 두 과목 이상이 겹치면 충돌"

### 감지 방법

모든 (강의실, 교시) 쌍에 대해 해당 슬롯을 쓰려는 과목 수를 센다.

```
K106, 3교시 ← 과목A, 과목B
→ 2건 이상 = 충돌! (빨간색 표시)
```

### 세 가지 충돌 유형

1. **요청 vs 요청**: 아직 배정 안 된 요청끼리 겹침
2. **요청 vs 배정**: 미배정 요청과 이미 배정된 과목이 겹침
3. **배정 vs 배정**: 배정된 과목끼리 겹침

### 예외: 같은 과목의 분반

```
과목A-01 → K106, 3교시
과목A-01+2 → K106, 3교시   (같은 과목의 분반)
→ 충돌 아님 (base key가 동일)
```

## 7. 빈 강의실 판정

> "이 교시에 이 방을 쓸 수 있는가?"

모든 필요 교시에 대해 아래 조건을 **전부** 통과해야 "빈 방":

```
 빈 방 = (시간표에 수업 없음  OR  해제된 슬롯)
       AND 다른 시험에 배정되지 않음
```

하나라도 걸리면 사용 불가.

---

## 8. 수급 현황 계산

### 공급 (빈 강의실 수)

교시별로, 용량 기준별로 빈 강의실을 센다.

```
3교시, 전체 강의실 중:
  K106: 수업 중 (해제 안 됨) → 제외
  K201: 비어 있음           → +1
  K302: 해제됨              → +1
  K401: 이미 시험 배정됨    → 제외
→ 공급 = 2
```

용량 필터도 적용된다 (30석+, 50석+, 80석+, 100석+).

### 수요 (미배정 요청 수)

아직 배정 안 된 강의실 변경/분반 요청이 각 교시에 몇 건인지 센다.

```
3교시에 시험 필요한 미배정 요청: 과목B, 과목C, 과목D
→ 수요 = 3
```

### 부족분

```
부족분 = max(0, 수요 - 공급)
```

| | 3교시 | 4교시 | 5교시 |
|---|---|---|---|
| 빈 강의실 | 2 | 5 | 8 |
| 미배정 수요 | 3 | 2 | 1 |
| **부족분** | **1** | 0 | 0 |

부족분이 양수면 해당 교시에 방이 모자란다는 뜻.

## 9. 분반 완료 판정

> "분반으로 나눈 과목이 충분히 배정되었는가?"

### 기존 유지 분반

원래 강의실을 유지하면서 추가 강의실을 배정 → **즉시 완료**

### 기존 미유지 분반

원래 강의실을 포기하고 새 강의실들로만 배정 → **수용 인원 검사**

```
수강생 = 120명
배정된 강의실: K201 (60석) + K302 (80석) = 140석
140 ≥ 120 → 완료
```

```
수강생 = 120명
배정된 강의실: K201 (60석) = 60석
60 < 120 → 미배정 (추가 배정 필요)
```

**공식**: `배정 강의실 수용인원 합 ≥ 수강생 수 → 완료`

(같은 강의실이 중복 배정되면 수용인원은 한 번만 합산)

## 10. 강의실 가동률

> "이 날 강의실이 얼마나 활용되고 있는가?"

```
전체 슬롯 = 강의실 수 x 15교시
사용 슬롯 = 시간표에서 수업이 있는 슬롯 수 - 해제된 슬롯 수

가동률(%) = (사용 슬롯 / 전체 슬롯) x 100
```

### 예시

```
강의실 10개, 총 슬롯 = 10 x 15 = 150
수업 있는 슬롯 = 85개
해제된 슬롯 = 5개

가동률 = (85 - 5) / 150 x 100 = 53.3%
```

해제가 많을수록 가동률이 내려간다 (시험 때 비워진 방이 많다는 의미).

## 11. 히트맵 색상 기준

잔여 슬롯 히트맵의 색상은 빈 강의실 수에 따라 결정된다:

| 빈 강의실 수 | 색상 | 의미 |
|---|---|---|
| 0개 | 빨강 (진) | 빈 방 없음 |
| 1~3개 | 빨강 (연) | 심각한 부족 |
| 4~10개 | 노랑 | 보통 |
| 11개+ | 초록 | 여유 |

## 12. 과목 키 중복 처리

같은 과목명+분반이 여러 행에 있으면 키가 겹친다. 이때 **행 번호**를 붙여 구분한다.

```
과목A-01 (5행)  →  과목A-01#5
과목A-01 (10행) →  과목A-01#10
```

분반 추가 배정 시에는 **+N** 접미사를 사용한다:

```
과목A-01      (첫 번째 배정)
과목A-01+2    (두 번째 배정)
과목A-01+3    (세 번째 배정)
```

## 14. 검수 큐 조건

자동 판정이 안 되어 사람이 확인해야 하는 5가지 경우:

| 조건 | 이슈 | 의미 |
|------|------|------|
| SKIP 분류 | 분류 미확정 | 스케줄 없음, 강의실 코드 없음 등 자동 분류 불가 |
| 시험일자 없거나 범위 밖 | 시험일자 누락/범위 밖 | 어느 격자(시트)인지 모름 |
| 교시 추출 실패 | 시험교시 정보 없음 | 몇 교시인지 모름 |
| 변경/분반인데 빈 방 없음 | 배정 후보 없음 | 해당 교시에 옮길 강의실이 0개 |
| NORMAL_EXAM + 시험교시 미명시 + 수업 2교시+ | 부분해제 확인 필요 | 자동 부분해제 불가, 수동 해제 필요 |

## 요약 — 한눈에 보는 핵심 산수

```
교시 시각      = 8 + 교시번호
시험 교시      = 시작교시 ~ 종료교시 (연속 범위)
부분해제 교시  = 수업 교시 - 시험 교시
충돌           = 같은 (강의실, 교시)에 2건+
빈 강의실      = 수업 없음(또는 해제됨) AND 배정 없음
부족분         = max(0, 미배정 수요 - 빈 강의실)
분반 완료      = 배정 강의실 수용인원 합 ≥ 수강생 수
가동률(%)      = (수업 슬롯 - 해제 슬롯) / (강의실 수 x 15) x 100
```
