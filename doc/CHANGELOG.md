# Changelog

## v2.5.0 (2026-04-06)

### 다주차 매핑 + 안정성 (코드 리뷰 반영)

- **[Medium] 다주차 시트 매핑**: `date_to_sheet` 직접 매핑 도입. 동일 요일 시트가 여러 개여도 날짜별 정확한 시트 참조. `_resolve_sheet()` 헬퍼로 전체 통일 (`DAY_TO_SHEET[` 직접 참조 0건)
- **[High] 기존 강의실 확정 시 교시 유실 방지**: `periods=[]`이면 슬롯에서 폴백, 그래도 없으면 버튼 비활성화
- **[Medium] 분반(기존 미유지) 자동 해제**: `keep_orig=False` 분반도 원래 강의실 자동 해제 대상에 포함
- **[Medium] 검수큐 분반 검색 기준**: `min_cap`을 `students/2` → `0`으로 변경 (분반은 수용인원 제한 없이 후보 검색)
- **[Medium] 스케줄 파서 공백 허용**: `화 4 ~ 5 ( K106 )` 등 공백 변형 정상 파싱

## v2.4.0 (2026-04-06)

### 분반/해제/데이터 대폭 개선

**배정 모드 확장**
- ROOM_CHANGE/ROOM_SPLIT 모두 3가지 배정 방식 선택 가능: 이동 / 분반(기존 유지+추가) / 기존 강의실 유지
- 분반 시 "기존 강의실 유지" 체크박스로 유지/미유지 전환, `keep_orig` 상태 저장 및 검증 반영
- 분반 다중 배정(+N 키), 수용인원 합산 기반 자동 완료 판정
- 이동 모드에서 수용인원 < 수강생 시 배정 차단

**자동 해제**
- NO_EXAM: 강의실 전체 교시 자동 해제 (핑크 표시)
- NORMAL_EXAM: 시험 교시 < 수업 교시일 때 미사용 교시 자동 부분 해제
- ROOM_CHANGE: 원래 강의실 자동 해제 (기존)
- 모두 구조화된 숫자 컬럼 기반 판단 (요청사항 텍스트 미참조)

**데이터 정합성**
- 수업시간표(콤마 구분)에서 시험일 요일 슬롯의 강의실로 `req.room` 자동 재설정
- 시간표에 없는 강의실(수업시간표 슬롯 기반)을 결과 검증(초록), 점유 현황, 해제 대상에 포함
- 분반 +N 키 판정: `rfind("+")+isdigit()`로 과목명 내 + 안전 처리
- 분반 키 생성: `max+1` 방식으로 덮어쓰기 방지
- 해제된 교시는 해제 드롭다운에서 자동 제외

**UI 개선**
- 배정 패널에 수업시간표 원문 표시
- 강의실 점유 현황 미니 격자 (시간표 외 강의실 포함, 수용인원 필터 제거)
- 교시별 수급 현황 (빈 강의실 vs 미배정 수요 vs 부족분)
- 배정 작업 현황 expander (분류별 그룹핑 + 개별 취소)
- 충돌 상세 보기 expander
- 해제 목록에 수용인원/수업명 표시, 접기 가능
- 검수 큐/완료 내역 엑셀 다운로드
- 결과 검증 엑셀에 범례 시트 추가
- 엑셀 데이터 다시 불러오기 버튼
- 변경 요청 미배정 색상: #FF8C00 → #FFD700 (갈색과 구분)

**안정성**
- 원자적 저장 (tempfile → rename), 손상 JSON 로그 경고
- StaleFileError 시 session_state 롤백
- 해제/자동해제 셀이 미배정 오버레이로 덮이지 않도록 방지
- NO_EXAM 슬롯 경로 교시 클램프 (0~14)

## v2.3.0 (2026-04-06)

### UI 구조 개선 + 정합성 수정 (코드 리뷰 반영)

**변경**: 4탭 → 5탭. "요청 + 배정" → **"배정 작업"** + **"배정 현황"** 분리.

- **[High] 섹션 헤더 건수 정합성**: 미배정/완료 헤더가 필터 적용 후 건수를 표시하도록 수정 (기존: 전체 건수)
- **[High] 검수 큐 필터 적용**: 검수 큐가 상단 분류/검색 필터를 따르도록 수정
- **[Medium] 엑셀 갱신 시 배정 동기화**: 엑셀 파일 mtime 변경 시 배정/해제 세션도 자동 재로딩
- **[Medium] 저장본 불러오기 mtime 갱신**: 불러오기 후 mtime을 갱신하여 이후 저장 시 StaleFileError 방지
- **[Medium] 용어 통일**: "검토필요" → "미확정"으로 통일 (Category.SKIP 라벨과 일치)
- **[Medium] 탭 분리**: 배정 작업(히트맵+배정+해제)과 배정 현황(검수큐+완료+취소+내보내기)을 별도 탭으로 분리

## v2.2.0 (2026-04-06)

### 안전성 개선 (코드 리뷰 반영)

- **[High] ROOM_SPLIT 충돌 감지 보완**: 분반 배정 시 추가 강의실만 충돌 집계에 포함되던 문제 수정. 원래 강의실도 `assignment_map`에 포함하여 배정 간 충돌을 정확히 감지
- **[High] 동시 작업 덮어쓰기 방지**: 저장 시 파일 mtime을 비교하여 다른 세션의 변경을 감지. 충돌 시 저장 차단 + 새로고침 안내 (`workflow_utils.StaleFileError`)
- **[Medium] 완료 지표 보정**: "완료" 항목 중 검수 큐에 해당하는 건수를 metric help 텍스트로 표시하여 진행률 과대 표시 방지
- **[Medium] 수동 배정 교시 경고**: 조건 직접 검색에서 선택 교시가 요구 교시와 다를 때 경고 메시지 표시
- **[Medium] 전체 초기화 확인 단계**: 체크박스 확인 후에만 초기화 버튼 활성화 (오클릭 방지)

## v2.1.0 (2026-04-06)

### 요청 + 배정 탭 통합

**변경**: 기존 5탭 → 4탭. "요청 현황" + "배정 작업" → **"요청 + 배정"** 단일 탭으로 통합.

**동기**: 요청 목록을 보다가 배정하려면 탭을 전환해야 했음. 한 화면에서 상태 확인 + 배정을 모두 처리하도록 개선.

- 진행 상황 요약 (완료/미배정/검토필요/전체) 실시간 metrics 추가
- 상태 필터 신규 추가 (완료/미배정/검토필요, 기본값: 미배정+검토필요)
- 히트맵을 접을 수 있는 expander로 이동
- 완료 섹션: NORMAL_EXAM(기존 확정) + NO_EXAM(미실시) + 배정완료 항목 통합 표시
- 개별 배정 취소 버튼 추가 (기존에는 전체 초기화만 가능)
- 내보내기 + 작업 이력을 하단 expander로 정리
- `compute_status(req, assignments)` 헬퍼 함수 추가

### 데이터 무결성 수정 (코드 리뷰 반영)

- **[High] 교시 범위 검증**: `resolve_needed_periods`가 0~14 범위를 벗어나는 교시를 클램프하도록 수정. 이전에는 범위 밖 교시가 "보이지 않는 배정"을 만들 수 있었음
- **[Medium] 캐시 갱신**: `cached_load`의 캐시 키에 파일 수정시간(mtime) 포함. 엑셀 파일이 바뀌면 자동으로 캐시 무효화
- **[Medium] 요청 키 충돌**: 동일 과목명-분반이 중복될 경우 행 번호를 붙여 고유화 (`과목명-분반#행번호`). 기존에는 조용히 덮어써짐
- **[Low] 히트맵 배정 반영**: 잔여 슬롯 히트맵이 수동 배정 결과를 반영하도록 수정
- 위 4건에 대한 회귀 테스트 7개 추가 (총 33개)

## v2.0.0 (2026-04-06)

### 프로젝트 재설계 — 수동 배정 지원 도구

**컨셉 전환**: 자동 배정 제거 → 시각화로 사람의 수동 배정을 가속화

- `exam_room_matcher.py` 제거 → `data_loader.py`로 대체 (파싱/분류 전용)
  - 자동 배정 로직 전부 제거 (auto_assign, apply_colors 등)
  - 결과 엑셀 생성 제거
  - 남은 것: parse_schedule, load_requests, classify_requests, load_timetable
- `run.py` 제거 → `run.bat`에서 직접 streamlit 실행
- `dashboard.py` 전면 재작성 (5탭, 업무 흐름 순서):
  1. **기존 시간표** — 원본 격자 (갈색/청록 구분)
  2. **요청 현황** — 전체 목록 + 필터/검색 + 요청사항 원문
  3. **배정 작업** — 잔여 슬롯 히트맵 + 과목 선택 → 빈 강의실 → 배정 + 수동 검색 + 결과 내보내기
  4. **결과 검증** — 요청+배정을 시간표에 오버레이 + 충돌 감지 (빨간 경고)
  5. **통계** — 분류별/일별/가동률
- 충돌 감지: 같은 강의실+교시에 2개+ 과목 겹침 시 빨간 셀 + 경고 배너
- 잔여 슬롯 히트맵: 교시별 수용인원 구간별(전체/30+/50+/80+/100+) 빈 강의실 수
- 클릭 배정: 과목 선택 → 빈 강의실 선택 → 배정 → session_state 저장 → CSV/엑셀 내보내기
- 요청사항 텍스트는 파싱하지 않고 원문 그대로 표시
